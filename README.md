# 🏥 入院管理アプリ (Hospital Record)

動物病院向けの入院患者管理 Webアプリ（プロトタイプ）。
iPad/iPhone/PCブラウザで動作。データは端末のブラウザ内（localStorage）に保存。

## 公開URL
https://somnus-vet.github.io/hospital-record/

## 主な機能
- 📊 **バイタル**: TPR/SpO₂を横軸=時刻のフローシート形式で入力。基準範囲外は赤ハイライト
- 💧 **In/Out**: 食事%(0/25/50/75/100ワンタップ) / 飲水 / 尿 / 便スコア(1-7色付)
- 💉 **治療**: 薬剤一括予定モーダルで複数薬剤×時刻×日数を一括作成
- 🔬 **検査**: 依頼/採取/結果待ち/結果あり の4状態カンバン、結果写真添付可
- 👨‍👩‍👧 **面会**: 飼い主の様子、頂き物、伝達事項を記録
- 🔔 **申し送り**: シフトチェンジ用ダッシュボード（直近12hスパークライン等）
- ⚡ クイック記録FAB、Undo (⌘Z)、ブラウザ通知

## 使い方
1. URLにアクセス → 右上「＋ 新規」で患者登録
2. または右上「⬆」ボタンで `demo_data.json` をインポート（症例ポチでデモ）

## デモデータの取り込み
1. このリポジトリの `demo_data.json` の中身をコピー
2. アプリで「⬆ インポート」→ 貼り付け → インポート
